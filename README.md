# FizzBuzz

## What is this project?

This program's objective is to get a number from 1 to 100
so that it can see whether or not it should print Fizz, Buzz, 
FizzBuzz, or the number inputted.

If the program detects that the number that was inputted is
divisible by 3, it prints Fizz.

If the program detects that the number that was inputted is
divisible by 5, it prints Buzz.

These two are able to work together so that if the number 
inputted is divisible by both 3 and 5, it prints FizzBuzz.

If the number is not divisible by 3 or 5, it simply prints
the number the user put in.
